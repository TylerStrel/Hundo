                            // Get the values from the UI
                            function getValues(){
                                
                                let startValue = document.getElementById("startValue").value;
                                let endValue = document.getElementById("endValue").value;

                                // Parse into Ints
                                startValue = parseInt(startValue);
                                endValue = parseInt(endValue);

                                if (Number.isInteger(startValue) && Number.isInteger(endValue)){
                                    let numbers = generateNumbers(startValue, endValue);
                                    displayNumbers(numbers);
                                }
                                else{

                                    alert("You must enter integers.");
                                }
                            }

                            // Generate numbers from startValue to the endValue
                            function generateNumbers(startValue, endValue){
                                
                                let numbers = [];

                                for (let index = startValue; index <= endValue; index++){

                                    numbers.push(index);
                                }

                                return numbers;
                            }

                            // Display the numbers and mark even numbers bold
                            function displayNumbers(numbers){
                                
                                let className = "even";
                                let templateRows = "";
                                for (let index = 0; index < numbers.length; index++) {
                                    
                                    let number = numbers[index];

                                    if (number % 2 == 0)
                                    {
                                        className = "even";
                                        
                                    }
                                    else{
                                        className = "odd";
                                    }
                                    // This does not render correctly with Prism see the source code
                                    templateRows += `${number}`
                                }

                                document.getElementById("results").innerHTML = templateRows;

                            }
getValues()

This function retrieves the values of the startValue and endValue from the UI. It parses the retrieved values into integers. If both values are valid integers, it calls the generateNumbers function to get an array of numbers between the startValue and endValue.The displayNumbers function is then called with the generated numbers to display them. If the values are not valid integers, it displays an alert warning the user to enter integer values.

generateNumbers()

This function accepts two integer parameters: startValue and endValue. It creates an empty array called numbers. Using a loop, the function generates a sequence of numbers from startValue to endValue, inclusive, and adds them to the numbers array. It returns the numbers array.

displayNumbers()

This function accepts an array of numbers as its parameter. It iterates over the given array, for each number, it checks if it is even or odd. Depending on whether the number is even or odd, it assigns a class name (even or odd) to that number. It constructs a string of HTML table rows with the numbers and their respective class names. Even numbers are given the "even" class, and odd numbers are given the "odd" class. Finally, it updates the inner HTML of an element with the id results with the constructed string, displaying the numbers in a table format with even numbers highlighted differently from odd numbers (presumably, given the class names).
